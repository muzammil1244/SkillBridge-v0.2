# 🕊️ SkillBridge


Skill Bridge 🚀
A full‑stack MERN job portal platform where Employers and Freelancers/Job Seekers can connect, apply for jobs, manage applications, chat live, and build resumes.



---
## 🔗 Live Demo

👉 **Deployed URL:** [https://your-live-link-here](https://skillbridge-inky.vercel.app/employer/home)


## 📌 Project Overview

Skill Bridge is a **two‑sided platform**:

### 👨‍💼 Employer Side

* Employer Registration & Login
* Post Jobs (CRUD)
* View Applications
* Filter & Search Candidates
* View Applicant Resume
* Live Chat with Applicants
* Profile Update

### 👨‍💻 Freelancer / Job Seeker Side

* User Registration & Login
* View Jobs
* Search & Filter Jobs
* Apply for Jobs
* Resume Builder & Resume Upload
* Live Chat with Employers
* Track Applications
* Profile Update

## 🛠️ Tech Stack (MERN)

### Frontend

* React.js
* Tailwind CSS 
* Javascript

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose)

### Authentication

* JWT (JSON Web Token)
* bcrypt.js

### Real‑Time Features

* Socket.IO (Live Chat)

### Deployment

* Frontend: Vercel 
* Backend: Render 
* Database: MongoDB Atlas

---


## ✨ Key Features

* 🔐 Authentication (Login / Signup)
* 👥 Role Based Access (Employer & Freelancer)
* 📄 Job Posting & Application System (CRUD)
* 🔍 Advanced Search & Filters
* 💬 Live Chat (Real‑Time)
* 📑 Resume Builder & Resume Viewer
* 📊 Application Tracking Dashboard
